Now We're Cooking!

Now We're Cooking is a mobile game for Android and iOS devices. The story takes place in a reality cooking TV show in which the host has turned all the food evil. You now must utilize their chef powers to combine food by swiping different ingredients together, before it's too late. Created using Corona SDK and Lua. Roles: Lead Producer, Programmer, Artist, Musician.


Solar Crysis

Solar Crysis is a twin-stick shooter where the player controls a "nanobot" whose job is to constantly repair a continuously breaking solar panel while protecting itself from bad guys. If a panel breaks the nanobot can no longer cross that panel, thus making the solar panel a dynamically changing playing field. Created using XNA 4.0 and C#. Roles: Lead Designer, Programmer, Artist

Math Performance Predictor

A web application which generates a bar graph which depicts predicted standardized math score statistics of students from 3rd – 8th grade depending on three variables which the user can modify. Users can create an account and store in results into their own personal database.


NSF Sortable

A web application which generates a unique list of four generated teacher name-cards which have different statistics and a picture to match. The user rates the teacher from most to least qualified by sorting the name-cards in order from best to worst. Results are stored into database.
